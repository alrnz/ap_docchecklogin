Known Problems
==============

Right now, a concurrent different login system might allow the DocCheck Dummy
user to log in. Although this will hopefully be fixed within the next two or
three releases, please choose a very complicated password for the dummy user.

You are always invited to report all kinds of bugs, request features or contribute
on our `GitHub`_ repository..

.. _GitHub: https://github.com/antwerpes/ap_docchecklogin
